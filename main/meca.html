INGENIERÍA MECATRÓNICA

DISEÑO & MANUFACTURA

Aplicación de principios de mecánica y diseño para transformar una bicicleta convencional en una unidad de procesamiento de alimentos autónoma.

TECNOLOGÍA APROPIADA

Basado en la filosofía de Maya Pedal (Guatemala), este sistema busca crear herramientas sostenibles y reparables. El objetivo es la autonomía energética mediante mecanismos que no dependen de la red eléctrica.

PROTOCOLO DE ENSAMBLAJE

FASE 1: ESTRUCTURA BASE

Se construyó una estructura de madera nivelada reforzada con escuadras metálicas para soportar las fuerzas dinámicas del pedaleo. La bicicleta se fijó elevando la llanta trasera para permitir su giro libre.

FASE 2: SISTEMA DE TRANSMISIÓN

El movimiento se transfiere por fricción desde la llanta a un eje primario. Se utiliza la Gearbox REV Ultra 90° para redirigir el torque verticalmente hacia el vaso.

FASE 3: INTEGRACIÓN CAD

Se diseñó e implementó un adaptador personalizado (CAD) para conectar el eje de salida de la caja de engranajes con el cople del vaso, eliminando holguras mecánicas.

RESULTADO DE INGENIERÍA

El prototipo valida que una idea simple puede convertirse en una solución funcional mediante la alineación precisa de ejes y el uso de componentes de transmisión. Se logró una velocidad suficiente para licuar sin vibraciones críticas.
